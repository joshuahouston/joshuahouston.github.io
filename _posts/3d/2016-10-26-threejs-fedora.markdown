There are many services out there that allow you to display your 3D models through the web, but they usually require a click+load time. Three.js is a customizable way to have a 3D model on the page as simple as a picture.

I bought a hat when I was in Cuba. I scanned it with 123D Catch, then modeled it in Blender. I exported it from Blender to a JSON object. JSON stands for JavaScript Object Notation.
