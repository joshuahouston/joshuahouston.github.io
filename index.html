Yesterday's Kayak

Updated 11/11/2014. The thing I like about this map is that you can rotate it and line it up with the direction of travel. It makes a lot more sense when you can picture the path you were on with the memory of where islands and whales were. If you notice Whale Island and Kayak Island have no form, it's because Mapbox updates its coastline very infrequently. I've made hundreds of changes to the coastline around Sitka and during the last coastline update I hadn't quite figured out how to tag islands.
